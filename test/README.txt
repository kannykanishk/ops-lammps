data.test.txt - Data file, including position and normal vectors which were randomly generated for 60 atoms

in.test.txt - LAMMPS input file

dumpOPS_forces - Calculation of force vectors of each atom

dumpOPS_torques - Calculation of torque vectors of each atom

dumpOPS_normals - Verification of stored normal vectors of each atom

log.lammps - LAMMPS log file

In terminal, ~/lammps/build/Debug/lmp < in.test.txt was entered to execute the input file. NOTE: The pathway of the build file with depend on your system!
